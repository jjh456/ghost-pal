# Ghost Pal

A quick, mobile-friendly tool to sit on a desk next to the PC and narrow down
the ghost type in Phasmophobia based on evidence and behavior observed during
a contract.

Tap the evidence and behavior traits you've confirmed, and the list of
possible ghosts updates live. No build step, no accounts, works offline
once loaded.
